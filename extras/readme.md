# Extras

> This folder contains a set of auxiliary, or "extra" files used in the development of this library

<br>

### Files

1. [*Chain plotter (Kamda-Kawai algorithm)*](/extras/Chain%20Plotter%20(Kamada-Kawai)): Contains the python script used to plot the nodes' positions in real time.
2. [*RX-Distance relation*](/extras/RX-Distance%20relation): This is a set of python, csv and matlab files used to plot the relationship bewteen the RX power and distance measured between 2 devices.
