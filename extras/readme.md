# Extras

> This folder contains a set of auxiliary, or "extra" files used in the development of this library

<br>

### Images

The [*Images*](/extras/images) folder contains all of the pictures & images used in the 'readme' files all across the repo.

<br>

### Files

1. [*2D Positioning*](../example/2D%20Positioning): This python app receives via Wi-Fi the relative positions between 1 tag & 2 anchors and plots them in real time.  

2. [*Chain plotter (Kamda-Kawai algorithm)*](/extras/Chain%20Plotter%20(Kamada-Kawai)): Contains the python script used to plot the nodes' positions in real time.

3. [*RX-Distance relation*](/extras/RX-Distance%20relation): This is a set of python, csv and matlab files used to plot the relationship bewteen the RX power and distance measured between 2 devices.

4. [*AI Integration*](/extras/AI%20Integration): Python app that reads the measurements made in each cycle and converts them into a *.csv* file. That file is then used to train a tinyML model.

<br>
(Each folder contains its own readme file to explain the contents in full detail)
